"# Next-Gen-Pharma

A polished corporate website scaffold for the Next Gen Pharma brand, centered on a modular and scalable front-end structure.

## Project structure

- index.html — main entry page and hero/innovation content
- assets/css/base.css — shared reset and design tokens
- assets/css/sections.css — component styles for the website section
- assets/js/main.js — lightweight interactive behavior
- assets/images/innovation-visual.svg — visual asset for the innovation section

## Preview locally

Open index.html in a browser, or run a simple local server from the project folder.
